Different Functions of the Application

  • REGISTER

    To access the application must be registered with it. To finish the process has to complete a registration form by inserting the fields shown in the following image.

  • LOGIN

    From the Home screen to identify a link shown. They redirect the user to an interface where you insert your credentials to start using the software.

  • INTERRUPTION

    allows creating interruptions in the tasks

  • CREATE NEW TASK.

    In the top navigation bar is always available a link to create a task.

  • GTD ACTIONS.

    in this menu you will find the different gtd actions.

  • Search.

    Search to find tasks by task name or by state.

  • CALENDAR.

    Uses a calendar to put your work on a specific day

  • STATISTICS.

    statistics to look at the productive and unproductive time produced
